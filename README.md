# ProiectTW25CraciunIulian
Clasificare si Numar de Inregistrare pentru Documente Electronice (integrare Dropbox)

Proiectul reprezinta o aplicatie web destinata gestionarii documentelor electronice.
Scopul ei este:
	-stocarea documentelor digitale în categorii (ex: contracte, facturi, acte personale)
	-generarea unui numar unic de înregistrare pentru fiecare document
	-asocierea documentelor cu un link extern generat prin integrarea cu Dropbox
	-afisarea si gestionarea documentelor într-o interfata moderna (SPA)

Aplicatia este formata din:
	-Backend RESTful
	-Frontend SPA
	-baza de date
	-integrare cu Dropbox API

  Node.js + Express.js, Sequelize ORM pentru accesarea bazei de date, Dropbox API v2 pentru integrarea cu serviciul extern, Multer pentru upload de fisiere

  Motiv: Setup simplu, cod clar, suport excelent pe macOS. Acest proiect fiind realizat pe un MacOS.
